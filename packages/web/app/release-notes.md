# Release Notes

## Version 0.1.0 - September 2025

### New Features
- **Web Interface**: Launch of Lace's web-based AI coding assistant
- **Multi-Provider Support**: Compatible with Anthropic Claude, OpenAI, LMStudio, and Ollama
- **Event-Sourcing Architecture**: Resumable conversations across restarts
- **Tool System**: 11 built-in tools for file operations, system commands, and workflow management
- **Project Management**: Create and manage multiple coding projects
- **Session Management**: Organize conversations within projects

### Developer Experience
- **Modern Stack**: Built with TypeScript, React 19, and React Router 7
- **Component Library**: DaisyUI-based design system with dark/light themes
- **Real-time Updates**: Live conversation streaming and tool execution
- **User Approval System**: Configurable approval policies for tool execution

### Getting Started
Welcome to Lace! This release includes the foundational features for AI-assisted coding. Create a project, start a session, and begin collaborating with your AI assistant.

For documentation and support, visit the docs section in the application.
