# AnimatedCarousel

AnimatedCarousel component

## Usage

```tsx
import AnimatedCarousel from '@/components/ui/AnimatedCarousel';

// Basic usage
<AnimatedCarousel />;
```

| `autoScroll`     | `boolean`           | Optional | -           |

| `itemsPerView`   | `{`                 | Optional | -           |

| `tablet`         | `number`            | Required | -           |
| `desktop`        | `number`            | Required | -           |
