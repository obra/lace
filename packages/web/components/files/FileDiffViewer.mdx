# FileDiffViewer

### Purpose
A comprehensive file difference viewer organism that provides professional-grade code comparison capabilities. Supports side-by-side and unified diff views with syntax highlighting, line numbers, and interactive features for code review and version control workflows.
### When to Use

## Usage

```tsx
import FileDiffViewer from '@/components/ui/FileDiffViewer';

// Basic usage
<FileDiffViewer />
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `diff` | `FileDiff` | Required | - |
| `viewMode` | `'side-by-side' | 'unified'` | Optional | - |
| `showLineNumbers` | `boolean` | Optional | - |
| `showFullFile` | `boolean` | Optional | - |
| `maxLines` | `number` | Optional | - |
| `onCopy` | `(content` | Optional | - |
| `className` | `string` | Optional | - |
